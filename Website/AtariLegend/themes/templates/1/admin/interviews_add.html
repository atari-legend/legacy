This is where the actual interview is inserted in the DB. Overhere you'll notice 3 main text boxes.

- Intro : This is where you'll write a little introduction to the interview. This text is placed in the boxes of the main interview section of AL (and in the search results)

- Chapters : This is a section especially created for chapters. Its a bit more complex. You just add as many chapters as you like. eg : Introduction, It started back then...These chapters should have some sort of hotspot. Therefor you should always select each chapter and press the hotspotsource button. It also necesarry to give each chapter a unique number (behind the # sign). This chapter can be link to a certain place in the actual interview section by selecting the place and pressing the hotspottarget button. And ofcourse adding the appropriate number.

- Interview : This is where you'll add the actual interview text.

Remember that you have a lot of freedom regarding layout issues. Even the "enters" you place in a text field are translated to the AL page, so make something nice of it, will you? :-)
