Let's say there are many different documents of the game Xenon in our database. Some of these docs belong to a game doc disc (like a sewer doc disk), other documents are linked directly to the game itself, like the manual. The first example would belong to the category 'Game Doc Disk' while the latter would belong to the 'Game Doc' category.

DOC CATEGORY

In this section you can add or edit a doc category which can be linked to a document. To edit an existing doc category, look up the name in the dropdown box. To add a new doc category, use the input field and press 'add'
