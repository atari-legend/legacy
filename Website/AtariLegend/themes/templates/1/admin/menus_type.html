MENU TYPE

In this section you can add or edit a menu type which can be linked to a menu set. To edit an existing menu type, look up the name in the dropdown box. To add a new menu type, use the input field and press 'add'. The following types should be setup (with these ID's) for this section to work properly :

1 - Game
2 - Demo
3 - Tool
4 - Music
5 - Picture
6 - E-zin/doc
