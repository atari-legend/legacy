STONISH INTEGRATION

Stonish.net is a website/project created by fellow member Bruno Mathieu. This website will ultimately be replaced or exists next to AL. However, its data will become part of AL. On this page the integration of the Stonish website at the startup of the AL project is explained step by step.


  • Make sure al necessary tables have been created or updated by performing all Database update scripts before you continue
  • Download from Stonish and upload into AL db the following tables :
    • stonish_allalliance
    • stonish_allcontent
    • stonish_allmenus
    • stonish_faq
    • stonish_infoteam
    • stonish_legend
    • stonish_namealliance
    • stonish_namemenus
    • stonish_nameteam
    • stonish_news
    • stonish_people
    • stonish_software
    • stonish_state
    • stonish_type
  • Run the integration script
  • Delete the stonish tables from the AL db - BE CAREFUL
